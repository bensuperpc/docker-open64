#### Install
You need Linux distribution like Ubuntu or Manjaoro

```sh
git clone https://github.com/Bensuperpc/docker-open64.git
```
#### Build
```sh
cd docker-open64 && make linux/amd64
```
#### Usage

```sh
./compiler.sh opencc hello.c
```
